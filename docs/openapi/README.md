# Версионируемый OpenAPI-контракт

В этой папке хранятся закоммиченные снапшоты OpenAPI.

- `openapi.v1.json` — актуальный снапшот публичного API-контракта.
- Обновляйте его при изменениях API-контракта:
  ```bash
  uv run python scripts/export_openapi.py --output docs/openapi/openapi.v1.json
  ```
- CI проверяет, что снапшот синхронизирован с кодом.
- При релизе по тегу файл публикуется как release-артефакт.
